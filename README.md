# Chat GPT clone

## Install

- Clone repository

```shell
 git clone https://github.com/cavaldos/ChatGPT.git
```

```shell

 npm install # pnpm install
```

```shell
 npm run dev # npm run dev
```

## Docker 
 - build docker
```shell
docker build -t chat-gpt .
```
 - run

``` shell
docker run -dp 3000:3000  chat-gpt .
